Working on buses

Click on the Files on the tool bar
All files will be displayed and click on the Buses
The list of all buses will be shown as above


Adding New Bus

To add a new bus click on the ADD NEW button
on the table Above
a window for adding a new bus will appear as shown above
Enter all the required details on the fields
Click on the date buttons to adjust the dates
Click on add to add the bus
Cancel to exit the process
And Clear to start a fresh

Updating the Buses
Bus RegNo and BusNo can't be changed

To update the bus, Click on the bus you want to
update on the table
Then click on the update button
A Window for updating the the bus will appear
During update you can Edit or Delete the bus

To edit make the new changes and click on update
To Delete click on the Delete button
You can search the bus to be updated by entering
the bus number and Clicking on search.
The REFRESH button will update the table
