Bank System Menus

You will Find the following menus in Bank System (some menus are always same in all applications). Click on a menu for further information.

Menu Name Menu Detail
File menu for new Account Printing etc.
Edit menu for Edit & Search.
View menu for Viewing Customers.
Options menu Contain different Options of Program.
Window menu for Closing Child Windows.
Help help menu (also contain copyright information).

File
Top of the Page
New Account Create a new Customer Account.
Print Customer Balance Print Customer Balance Sheet.
Quit Bank System ? To exit From Bank System.

Edit
Top of the Page
Deposit Money Store Record of Deposit Money in Bank.
Withdraw Money Store Record of Withdraw Money From Bank.
Delete Customer Delete Any Customer From BankSystem.
Search Customer Search Customer By No. From BankSystem.
Search Customer By Name Search Customer By Name From BankSystem.

View
Top of the Page
View One By One View Single Customer's Account Status Detail.
View All Customer View All Customer Account Status Detail.

Option
Top of the Page
Change Background Color Change the Back Color of Program.
Change Layout Style Change the Layout of Program Completely.
Apply Theme Apply Different Kinds of Looks to Program.

Window
Top of the Page
Close Active Window Close the Active Child Window.
Close All Windows Close All the Child Windows.

Help
Top of the Page
