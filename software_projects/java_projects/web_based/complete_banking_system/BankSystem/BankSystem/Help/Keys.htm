Bank System Shortcut Keys

For the Menus, Shortcut Keys are given below.

Menu Name Shortcut Key of Menu
File Alt + F (To Open File Menu).
Edit Alt + E (To Open Edit Menu).
View Alt + V (To Open View Menu).
Option Alt + O (To Open Option Menu).
Window Alt + W (To Open Window Menu).
Help Alt + H (To Open Help Menu).
