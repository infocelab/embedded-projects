How to run this program :

 

In Windows System just double click at “MyUniversity_1_5.jar” Or “MyUniversity_1_4.jar” up to your JRE version

In Other System you simply type “ java –jar MyUniversity_1_5.jar” Or “ java –jar MyUniversity_1_4.jar”

 

This distribution is ready to use in Windows System but

If you use another system please create new Database that those system supported

And adjust the file “conf/conf.txt”

If you use my database (MS Access) you must create DNS before

Go to Controlpanel à AdministrativeToolsàData Sources (ODBC)

This dialog will appear !!

 

 

Click “Add” button at the right

 

 

Select “Microsoft Access Driver (*mdb)” then click “Finish” button

 

 

At Data Source Name field : Type “myDB” then click “Select..” button

 

 

Now you have to find myDB.mdb by youself !

Click OK OK and OK
