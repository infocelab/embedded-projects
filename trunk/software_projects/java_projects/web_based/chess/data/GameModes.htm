• Local Mode:
1. Select local mode by clicking on the Local radio button (by default game is in Local mode.
2. Click on start button to start the game
 
  • Remote Mode (Currently support is only for intranet network):
1. Application in both the player machines should be running (started).
One of the player (say player 1) can:
2. Select Remote mode by clicking on the Remote radio button
3. Type the machine name in the Textbox for the machine name
4. Click on connect.
5. A message will appear in the other player(player2) 's machine asking if the player
wants to accept the connection request.
6. If no is clicked the player1 will be informed about the refusal
7.If yes is clicked, the player 1 will be informed about the acceptance.
8. Player1 can start the game by clicking on start.
9. Player 2 will then be informed about the start of the game
10. Player1 will play white and can only move white player and that too only when his turn comes
11. Player2 will play black and can only move black player and that too only when his turn comes

Any player can reset the game, which will be informed to the other player. To connect to another machine, the game must be reset if it in on and then after typing the machine name, the connect button should be clicked. Connection with the previous machine is discontinued and the other machine is informed.
If one player exits from the game, the other player is informed.
