Working on Routes

Click on the Files on the tool bar
All files will be displayed and click on the Routes
The list of all Routes will be shown


Adding New Route
To Add new route click on the ADD ENTRYbutton
Registration window will appear

Enter all the required details on the fields
Click on the date buttons to adjust the dates
Click on add to add the bus
Cancel to exit the process
And Clear to start a fresh

Updating the Routes
Do not change Route Number
To update the Employee, Click on the Route you want to
update on the table
Then click on the UPDATE button
A Window for updating the the Routes will appear

During update you can Edit or Delete a route

To edit make the new changes and click on update
To Delete click on the Delete button
You can search the Route to be updated by entering
the Route number and Clicking on search.
The REFRESH button on the table will update the table
